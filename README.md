Aplicação em ReactJs para operações de rotas de entrega.

## Instruções: Em breve instruções sobre como usar

## Funcionalidades:

A interface foi desenvolvida focando em processos de operações de logística para de rotas de entregas.

### 🔐 Autenticação: Sistema de login validado no banco de dados, com níveis de acesso (ex: Gestor e Entregador).

### 📍 Gestão de Rotas: Gestor cadastra as rotas fazendo upload de um arquivo .xlsx com as informações daquela viagem e os pontos de entregas, é possivel alterar dados como o motorista e a placa do carro.

### 👥 Gestão de Usuários: Módulo administrativo dedicado ao controle do ciclo de vida dos colaboradores na plataforma. - Em evolução

### 📦 Comprovante de Entrega: Funcionalidade para registro e confirmação de entregas realizadas, Motorista vê a lista das entregas, e faz o registro da forma de pagamento e colhe da assinatura do cliente na tela do celular, ao finalizar a entrega as imagens são armazendas.

### 📱 Interface Responsiva: Design adaptável para dispositivos móveis, facilitando o uso. - Em evolução

### ⚡ Feedback em Tempo Real: Notificações e alertas visuais integrados para erros de login ou confirmações de ações com sucesso. - Em evolução

### 📊 Dashboard de Controle: Painel para acompanhamento do status das entregas (Pendente, Em Rota, Concluído). - Próximas Implementações


## 🛠️ Tecnologias Utilizadas
React.js: Biblioteca principal para construção da interface.

Chakra UI: Framework de componentes para um design moderno e acessível.
